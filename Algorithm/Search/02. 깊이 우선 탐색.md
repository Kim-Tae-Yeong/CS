# 깊이 우선 탐색
## 깊이 우선 탐색(Depth - First Search, DFS)
깊이 우선 탐색은 **그래프에서 깊이를 우선적으로 탐색**하는 방법이다.

즉, **루트 노드(혹은 다른 임의의 노드)에서 시작해서 다음 분기로 넘어가기 전에 해당 분기를 완벽하게 탐색**하는 방법이다.

이는 주로 **경로의 모든 경우를 탐색**할 때 사용한다.
